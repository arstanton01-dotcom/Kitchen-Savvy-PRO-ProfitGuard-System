# Vendor Strike Logic  
Kitchen Savvy Pro – ProfitGuard System  

## Purpose
Define the logic for penalizing non-compliant vendors and enforcing accountability.

## Strike Rules
1. Minor Discrepancy (≤ 2% deviation)  
   - Warning flag  
   - Log event  

2. Major Discrepancy (> 2% deviation)  
   - Escalate to operator  
   - Add to recovery queue  

3. Repeat Offender (> 3 strikes/month)  
   - Highlight in vendor dashboard  
   - Consider renegotiation or removal  

## Outputs
- Vendor scorecard  
- Recovery action tracking  
- Automated alert notifications
