## Purpose
Define the rules that automatically trigger system workflows.

## Workflow Triggers
1. Invoice Discrepancy > 0  
2. Credit Ageing  
   - 7 days: At Risk  
   - 14 days: Critical  
3. Cost % Variance Threshold Breach  
4. Vendor Performance Decline  

## Automated Actions
- Queue for review  
- Notify operator via dashboard/email  
- Generate recovery request
