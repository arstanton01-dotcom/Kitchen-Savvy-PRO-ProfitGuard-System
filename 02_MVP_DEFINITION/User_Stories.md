# MVP User Stories

- As an owner-operator, I want to see weekly margin variance so I can take immediate corrective action.
- As a manager, I want vendor invoice errors highlighted so I can recover lost margin.
- As a multi-unit operator, I want a simple dashboard showing store comparison.
- As a CFO, I want confidence-scored alerts so I can prioritize recovery tasks.
- As a user, I want my data displayed cleanly with branded visuals and consistent formatting.
